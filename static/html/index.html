Endpoints for PetStation

USERS API

HTTP PATH DESCRIPTION
GET / Get all users
GET /user/:id Get a user
POST /user/register Register a user
PATCH /user/update/:id Update a user
POST /login Login a user
DELETE /users/delete/:id Delete/remove a user

Products API

HTTP PATH DESCRIPTION
GET /products Get all products
GET /product/:id Get a product
POST /addProduct Register a product
PUT /product/update/:id Update a product
DELETE /products/:id Delete/remove a product

CART API

HTTP PATH DESCRIPTION
GET /user/:id/carts Get all carts for a user
POST /user/:id/cart Add an item to the cart
PATCH /user/:id/cart/:id Update an item in the cart
DELETE /user/:id/cart Remove an item from the cart
